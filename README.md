# City Explorer

**Author**: Calvin, Steven
**Version**: 1.0.0

## Overview
It provides location and weather information for locations.

## Getting Started
You can use GET to get weather data from /weather and location data from /location

## Architecture
It uses node, express, dotenv, and cors.
